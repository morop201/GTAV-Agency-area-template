This is to be used with LSPDFR to modify "agency.xml" located
in lspdfr>data>agency.xml by right clicking and selecting "edit".

Required stuff:
OpenIV - https://openiv.com/

This will make government agencies like San Andreas Highway Patrol spawn in San Andreas
Highways.

This is very useful when you're a GTA V LSPDFR mod creator.
